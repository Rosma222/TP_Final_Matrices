# TP_Final_Matrices

Código desarrollado en Lazarus con lenguaje Pascal, para demostrar de forma práctica operaciones de álgebra en matrices bidimensionales.

## 🚀 Descargar Programa Listo para Usar

No necesitas tener Lazarus instalado para probar el proyecto. Puedes descargar la versión ejecutable directamente desde el siguiente enlace:

* **[Descargar TP_Final_Matrices para Windows (Ejecutable) 📥](https://github.com/Rosma222/TP_Final_Matrices/releases/download/v1.0.0/matrices.exe)**

*(Nota: Descarga el archivo, dale doble clic y ejecuta. Si Windows muestra una advertencia de SmartScreen, haz clic en "Más información" y luego en "Ejecutar de todas formas").*
