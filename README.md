# TP_Final_Matrices

Código desarrollado en Lazarus con lenguaje Pascal, para demostrar de forma práctica operaciones de álgebra en matrices bidimensionales.

## 🚀 Descargar Programa Listo para Usar

No necesitas tener Lazarus instalado para probar el proyecto. Puedes descargar la versión ejecutable directamente desde el siguiente enlace:

* **[Descargar TP_Final_Matrices para Windows (Ejecutable) 📥](https://github.com/Rosma222/TP_Final_Matrices/releases/download/v1.0.0/matrices.exe)**

*(Nota: Descarga el archivo, dale doble clic y ejecuta. Si Windows muestra una advertencia de SmartScreen, haz clic en "Más información" y luego en "Ejecutar de todas formas").*

## 📋 Requisitos del Sistema

* **Sistema Operativo:** Windows 10 o Windows 11 (64 bits).
* **Dependencias:** No requiere instalaciones previas (es un ejecutable portátil autónomo).
* **Resolución de Pantalla:** Optimizada para 1024x768 píxeles o superior.

## 🕹️ Instrucciones de Uso

Sigue estos pasos para realizar operaciones con matrices en el programa:

1. **Definir Dimensiones:** Introduce el número de filas y columnas en los casilleros e ingresa los valores numéricos correspondientes para la Matriz A y la Matriz B, con las que vas a trabajar (por ejemplo, 3x3).
2. **Generar matrices:** Haz clic en el botón y se generaran ambas con valores aleatorios.
3. **Seleccionar Operación:** Elige la acción matemática que deseas realizar desde el menú de botones:
   * ➕ Sumar matrices
   * ➖ Restar matrices
   * ✖️ Multiplicar matrices
   * 📐 Calcular determinante o matriz transpuesta
4. **Ver Resultados:** El programa calculará la operación instantáneamente y mostrará la matriz resultante en la cuadrícula inferior.
